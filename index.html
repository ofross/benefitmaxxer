📚 How to export a CSV from your bank ⌄
Chase
  1. Sign in at chase.com
  2. Select the credit card account
  3. Click Download account activity (below the transaction list)
  4. Choose date range → file type CSV → Download
Amex
  1. Sign in at americanexpress.com
  2. Go to Statements & Activity
  3. Click Download (top-right of transaction list)
  4. Select Comma Separated Values (.csv) → Download
Citi
  1. Sign in at online.citi.com
  2. Select the card account → Account Activity
  3. Click Download → choose date range
  4. Select CSV format → Download
BofA
  1. Sign in at bankofamerica.com
  2. Select the card account → Transactions tab
  3. Click Download → choose date range
  4. Select Comma Separated Values (CSV) → Download
Capital One
  1. Sign in at capitalone.com
  2. Select the card account
  3. Click Download Transactions
  4. Choose date range → CSV → Download
Wells Fargo
  1. Sign in at wellsfargo.com
  2. Select the card account → Account Activity
  3. Click Download Account Activity
  4. Choose date range → Comma Delimited → Download

✦ Connect Copilot Money

Deploy worker/copilot-proxy.js to Cloudflare Workers, then paste the URL above. This is a stateless relay you control — it never logs or stores your token.

How to get your token
  1. Open app.copilot.money in a new tab and log in.
  2. Open DevTools: F12 or Cmd ⌥ I
  3. Go to the Network tab → filter for graphql.
  4. Click any transaction or reload the page to trigger a request.
  5. Click a graphql request → Headers tab.
  6. Find Authorization: Bearer … and copy everything after Bearer .

🔒 Your token is held in memory only. It is never written to disk, localStorage, or any server. It expires automatically when your Copilot session ends.
